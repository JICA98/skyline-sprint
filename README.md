# Skyline Sprint

An original open-source PS4 homebrew game built using the OpenOrbis PS4 Toolchain. It is an infinite side-scrolling platform runner designed for performance, compatibility testing, and fun.

## Features
- **Deterministic Procedural Level Generation**: Chunk-based infinite side-scroller.
- **Fixed Timestep (60 Hz)**: Physics and game simulation run independently of the frame rate.
- **Diagnostics Screen**: On-screen performance overlay showing memory usage, CPU/GPU frames, and controller status.
- **Platform Agnostic**: Compiles and runs both on standard desktop (SDL2) and PlayStation 4 (OpenOrbis).
